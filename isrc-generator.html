¿Qué es un ISRC?

El International Standard Recording Code es el código único e irrepetible que identifica cada grabación de audio o video musical en el mundo. Es el "DNI" de tu canción.

Sin ISRC, plataformas como Spotify, Apple Music o YouTube no pueden rastrear correctamente tus reproducciones ni pagarte regalías de forma precisa.

Cómo obtener ISRCs oficiales
1
Tu distribuidor (más fácil)
DistroKid, TuneCore, CD Baby y la mayoría de distribuidores asignan ISRCs automáticamente al subir tu canción. Sin coste adicional. Es la opción más sencilla para artistas independientes.
2
Tu PRO (más control)
SGAE (España), ASCAP/BMI (EE.UU.), SOCAN (Canadá). Registras tú mismo el código ligado a tu obra. Tienes control total sobre el ISRC y no dependes del distribuidor.
3
IFPI directamente
Si tienes sello propio, puedes solicitar un código de registrante a la IFPI de tu país. Esto te permite generar ISRCs ilimitados bajo tu propia identidad de sello.
Reglas importantes
!
Un ISRC por versión
La versión original, el remix, el acoustic version, el live version — cada uno necesita su propio ISRC. No reutilices el mismo código para versiones distintas.
!
Es permanente
Un ISRC asignado nunca cambia ni se reasigna a otra canción, aunque el tema se elimine de las plataformas.
!
Guarda un registro
Mantén una lista de todos tus ISRCs con el título del track y la fecha. Usa la sección "Mis ISRC" de esta herramienta para organizarlos.
